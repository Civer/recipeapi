Recipe

Endpoints for managing recipes

Base URL: http://APIServerIP/recipe/

METHOD ENDPOINT USAGE RETURNS
POST /recipe/create Used for creating new recipes or updating existing ones info
GET /recipe/get Used for getting a recipe based on ID recipe
GET /recipe/search Used for searching recipes based on params recipe[]


POST: /recipe/create

This method creates either creates a new recipe or updates and existing recipe based on the uuid

Request Parameters

Body Parameters

REQUEST BODY DATA VALUE TYPE VALUE
recipe JSON object Mandatory: A JSON Object in Open JSON Recipe Format. Optionally with recipe_uuid filled. If recipe_uuid is filled the uuid will used to perform a search in database and update existing object. If uuid is not filled a new recipe will be created.

Cookie Parameters

COOKIE NAME VALUE TYPE VALUE
token Bearer Token MandatoryA signed Bearer Token signed with your secret API key. Containing mandatory parameter user

Response Format

If a new recipe is created return parameter will be the uuid of newly created recipe. If recipe is updated a message of succesful update is returned. If an error occurs it will be returned.


GET: /recipe/get

This method returns a recipe based on uuid

Request Parameters

Path Parameters

PATH PARAMETER VALUE
recipe_uuid The UUID of the recipe. UUID is a 12 byte String returned after succesful creation of recipe.

Example: IP/recipe/get/5db1d72c1a6d732590c1a08d

Cookie Parameters

COOKIE NAME VALUE TYPE VALUE
token Bearer Token MandatoryA signed Bearer Token signed with your secret API key. Containing mandatory parameter user

Response Format

Returns the recipe in Open JSON Recipe Format O(J)RF


GET: /recipe/search

This method returns a recipe based on uuid

Request Parameters

Path Parameters

PATH PARAMETER VALUE
search_object Type of search. Possible parameters:
  • recipe_name
  • ingredient_name
  • ingredient_x_id
search_string String used in search

Example: IP/recipe/search/recipe_name&Apfelkuchen

Cookie Parameters

COOKIE NAME VALUE TYPE VALUE
token Bearer Token MandatoryA signed Bearer Token signed with your secret API key. Containing mandatory parameter user

Response Format

Returns the recipe in Open JSON Recipe Format O(J)RF
